# user-auth-system

## Software And Tools Used
* Apache Netbeans IDE 12.0
* Microsoft Access Database .accdb
* Java [JDK 15]
* Java GUI components

## Description
* It allows a user to login using a username and password.
* It allows a new user to register itself.
* It allows an existing user to change its password by asking for a username and old password.
* It allows an existing user to set a new username by asking for an old username and password.
* Username should not be more than 20 characters. Whereas the password should be between 4 to 10 characters long. To make this process secure, password is encrypted before storing it into the file.
